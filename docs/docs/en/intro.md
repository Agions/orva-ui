# Orva UI

Orva UI is a modern cross-platform React component library built for the Taro ecosystem.

## Introduction

Orva UI provides a comprehensive set of high-quality React components that work seamlessly across web, mobile, and mini-program platforms.

## Features

- 🎨 **Modern Design** - Clean, professional design system
- 📱 **Cross-Platform** - Works on web, mobile, and mini-programs
- 🚀 **Performance** - Optimized for fast rendering
- 🌐 **i18n Ready** - Built-in internationalization support
- 📦 **Modular** - Tree-shakeable, import on-demand
- 🎯 **TypeScript** - Full TypeScript support

## Installation

```bash
npm install orva-ui
```

## Quick Start

```tsx
import React from 'react';
import { Button } from 'orva-ui';

export default () => <Button>Hello Orva UI!</Button>;
```
